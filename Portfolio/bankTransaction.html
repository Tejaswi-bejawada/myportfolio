Bank Account Transaction System

The Bank Account Transaction System is a robust project developed using Spring Boot and tested with Postman.

Key Features:

  • Creating different types of bank accounts, such as savings, salaried, and joint accounts.
  • Facilitating secure money transfers between accounts.
  • Efficient withdrawal process for account holders.
  • Real-time bank balance inquiries to keep track of finances.
  • Updating and managing user details for account holders.
  • Providing options to delete user accounts when needed.

This project showcases my expertise in building complex web applications, integrating with databases, and ensuring data security for financial transactions.
