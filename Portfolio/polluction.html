Pollution Check App

The Pollution Check App is a project I developed using Angular and Spring Boot.

Key Features:

  • Check the Air Quality Index (AQI) of a specific location.
  • Retrieve temperature and humidity data for the same location.
  • Wishlist feature to save favorite cities for quick access.

This project aims to provide users with up-to-date environmental data and make it convenient to track and manage their favorite cities pollution levels.
