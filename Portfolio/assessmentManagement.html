Assessment Management System

The Assessment Management System is a project I developed using microservices architecture and Thymeleaf templating. It provides access for both administrators and employees, simplifying the assessment scheduling and management process.

Key Features:

  1. Employee Access:
    • Registration and login for employees using credentials.
    • Ability for employees to schedule assessments on available dates.
  2. Admin Access:
    • Comprehensive access for administrators to view and manage registered employees.
    • Access to view all scheduled assessments and manage assessment schedules.
